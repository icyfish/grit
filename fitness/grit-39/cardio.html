# 01. WARM-UP

  • Hip Opener 10s
  • Squat with Arm Reach O/H 15s
  • Dynamic Lunge with Rotation L&R 10s
  • High Knee Run OTS 10s
  • (Turn to face R side) Lateral Shuffle F&B 10s
  • Half Burpee 10s
  • Half Burpee Jump 10s
  • High Knee Run OTS 10s
  • (Turn to face L side) Lateral Shuffle F&B 10s
  • Pushup x1, Mountain Climber L&R x2 20s
  • Lateral Shuffle Alt F&B x1, Low 180o Squat Jump 35s
  • High Knee Sprint OTS 10s
  • Recovery

# 02. TABATA PYRAMID UP

  • A. Lateral Skater x3, Single-Leg Pushup x1 20s
  • Recovery 10s
  • B. Broad Jump x1, Burpee Jump x1, Jack Back x2 20s
  • Recovery 10s
  • C. Shuttle L&R 20s
  • Recovery 10s

TIP

SET 1: ABC

SET 2: AABBCC

SET 3: AAABBBCCC

# 03. SUPERSET KICKER

  • Pop Squat L&R (lower body)
  • Frog Jump, Shoulder Tap L&R (upper body)
  • On/Off Beat Kicker Sprint OTS
  • Recovery

TIP

SET 1: 2x 30-second intervals, 1x 10-second Kicker Sprint

SET 2: 2x 25-second intervals, 1x 15-second Kicker Sprint

SET 3: 2x 20-second intervals, 1x 20-second Kicker Sprint

# 04. REP CHALLENGE DROP SET

# SET 1

  • Power Lunge L&R x1, Burpee Jump x1 (6 reps)
  • Power Lunge L&R x1, Burpee Jump x1 (5 reps)
  • Power Lunge L&R x1, Burpee Jump x1 (4 reps)
  • Power Lunge L&R x1, Burpee Jump x1 (MAX EFFORT)
  • Recovery

# SET 2

  • Double V-Tap (6 reps)
  • Double V-Tap (5 reps)
  • Double V-Tap (4 reps)
  • Double V-Tap (MAX EFFORT)
  • Recovery

TIP

30s 内完成括号中的组数, 一个 SET 结束休息 30s.

# 05. TABATA PYRAMID DOWN

A: Lateral Skater x3, Single-Leg Pushup x1 B: Broad Jump x1, Burpee Tuck Jump x1, Jack Back x2 C: Shuttle L&R

TIP

SET 1: AABBCC SET 2: ABC

每个步骤 30s, 中间休息 10s.

# 06. CORE CHALLENGE
