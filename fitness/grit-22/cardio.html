# 01. WARMUP

  • Jog OTS
  • Jog OTS, Heels Up
  • High Knee Run
  • Shuffle Tap (x2) (L+R)
  • Shuffle Tap with Leg B (x2) (L+R)
  • Squat Burpee Jump
  • Pushup
  • Jumping Switch Climber (L+R)
  • Long Jump F, Shuffle B
  • Long Jump F, High Knee Run B
  • Recovery

# 02. CONTRAST TRAINING

# SET 1

  • High Knee Run (x8), Speed Squat Jack (x4)
  • Jumping Switch Climber (L+R)
  • High Knee Run (x8), Air Tap (x4)
  • Recovery.

# SET 2

  • Lateral Leap (x1), Block Jump (x1) (L+R)
  • Single-Leg Frog Jump (L+R)
  • Lateral Leap (x1) Tuck Jump (x1) (L+R)
  • Recovery

TIP

每个动作30s, 组间休息30s, 重复2轮.

# 03. 10/10 KICKER

# SET 1

  • Butt Kick, Wide Squat (x10)
  • Bear Crawl to Double Jack Pushup (x10)
  • KICKER: Long Jump F, High Knee Run B
  • Recovery

# SET 2

  • Tuck Jump Lunge (x10) (L+R)
  • Squat Burpee (x1), Run F+B (x10)
  • KICKER: Snowboarder (L+R)

TIP

每组一分钟, 20s休息, 重复2轮.

# 04. SUPERSETS

  • TEAM A: Squat Tuck Jump, TEAM B: Chimp Walk
  • TEAM A: Chimp Walk, TEAM B: Squat Tuck Jump
  • Recovery

TIP

做两轮, 第二轮AB组交换动作. 每个动作 30s, 轮次间休息 20s.

# 05. EVERY 30 SECONDS

# SET 1

  • Pushup, Half Squat, Speed Squat Jack (x8) * 3
  • Recovery 30s

# SET 2

  • Squat Burpee Jump with Clap O/H (x8) * 3
  • Recovery 30s

TIP

一轮, 每个动作 30s

# 06. COACH CALLS

  • Drop Squat
  • Plyo Sumo Squat Jump
  • Mountain Climber
  • Recovery

TIP

2分钟内3个动作轮流进行.

# 07. CORE CHALLENGE

  • Pike with Alternating Knee Bend (L+R)
  • Oblique Crunch (L+R)
  • Walking Hover
  • Recovery
