## 01. WARMUP

- Jog OTS - 15s 
- Jog OTS, Heels Up - 15s 
- High Knee Jog - 15s
- High Knee Run - 20s
- Squat Burpee - 30s
- Bronco - 25s
- Bronco (x1), Block Jump (x1) - 15s
- Lateral Leap (L+R) - 25s
- Squat Air Tap - 15s
- Recovery 
## 02. MONSTER SETS

- STATION 1
  - Bronco (x1), Block Jump (x1) 
- STATION 2
  - Squat Air Tap
- STATION 3
  - Lateral Leap (L+R)
- STATION 4
  - High Knee Run

:::
