## 02. CONTRAST TRAINING

#### SET 1

- High Knee Run (x8), Speed Squat Jack (x4)
- Jumping Switch Climber (L+R)
- High Knee Run (x8), Air Tap (x4)
- Recovery. 

#### SET 2

- Lateral Leap (x1), Block Jump (x1) (L+R)
- Single-Leg Frog Jump (L+R)

## 03. 10/10 KICKER

#### SET 1

- Butt Kick, Wide Squat (x10)
- Bear Crawl to Double Jack Pushup (x10)
- KICKER: Long Jump F, High Knee Run B
- Recovery

#### SET 2

- Tuck Jump Lunge (x10) (L+R)
- Squat Burpee (x1), Run F+B (x10)

## 04. SUPERSETS

做两轮, 第二轮AB组交换动作. 每个动作 30s, 轮次间休息 20s.
:::
